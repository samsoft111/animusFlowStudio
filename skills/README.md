# Diretório de Skills de Geração por IA (Temas & Plugins)

> [!IMPORTANT]
> Os ficheiros e instruções contidos nesta pasta **NÃO são configurações nem pertencem ao projeto AnimusFlowStudio** propriamente dito.
>
> Esta pasta serve exclusivamente como um repositório centralizado para armazenar **Instruções de IA (Skills/Prompts)**. Estes documentos são fornecidos ao motor de inteligência artificial durante o fluxo de criação para ditar a lógica, estrutura de ficheiros, design system e especificações técnicas de:
>
> 1. **Criação de novos Temas** (dentro da pasta `themes/`).
> 2. **Criação de novos Plugins** (dentro da pasta `plugins/`).
>
> Ao criar ou personalizar novos pacotes via IA, pode carregar estes ficheiros no editor para direcionar a geração automática da estrutura HTML, variáveis CSS e comportamento Javascript.
